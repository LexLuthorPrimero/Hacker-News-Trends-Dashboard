# Hacker News Trends Dashboard

Dashboard interactivo que muestra tendencias de Hacker News en tiempo real.

## Características

- Extracción de los 100 posts más votados
- Almacenamiento en SQLite
- Visualización de puntuación y comentarios
- Nube de palabras de títulos
- Actualización manual con un botón

## Instalación y uso

```bash
git clone https://github.com/LexLuthorPrimero/Hacker-News-Trends-Dashboard.git
cd Hacker-News-Trends-Dashboard
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
streamlit run app.py
```
